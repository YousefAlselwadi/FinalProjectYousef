Services

How I can help improve the quality of your web applications.

UI Test Automation

Creation of automated end-to-end tests for key user flows using tools like Playwright and Cypress. This helps catch regressions quickly when new features are released.

API Testing

Verification of REST APIs including status codes, response structures, error handling, and performance checks. Tools may include Postman and Playwright API testing.

Regression & Smoke Suites

Design and maintenance of automated test suites that can be run on every build or deployment to ensure stability and reliability.

Test Planning & Documentation

Writing clear test cases, organizing test data, and documenting results so that teams understand what has been tested and what still needs attention.

Database Validation

Using SQL queries to verify that data is stored and retrieved correctly, and that calculations match what users see in the UI.

Collaboration & Agile Testing

Working closely with developers, product owners, and other team members in Agile/Scrum environments to keep quality a shared responsibility.
