Personalized Banking Dashboard

UI, API, and database testing for a financial dashboard that surfaces real-time insights.

Overview

This project involved testing a personalized banking dashboard that helps customers track their spending, view balances, and receive tailored financial insights.

My Role

  • Designed test cases for key user journeys (login, viewing accounts, viewing insights).
  • Created automated UI tests using a modern testing tool.
  • Performed API testing for the services that powered the dashboard data.
  • Validated database records to ensure calculations and balances were correct.

Tools & Technologies

  • Playwright / Cypress for UI automation
  • REST API testing tools
  • SQL for database checks
  • Git and CI/CD integration for running tests on new builds

Key Outcomes

The testing effort helped detect several issues early, including incorrect balance calculations and edge cases with filtering transactions. Automated tests reduced manual regression time and improved confidence in releases.
